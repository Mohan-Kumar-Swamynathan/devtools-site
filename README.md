# DevTools.site

Free online developer tools - JSON formatter, Base64 encoder, JWT decoder, and 50+ more tools.

## Features

- 🚀 **Fast** - All tools run locally in your browser
- 🔒 **Private** - Your data never leaves your device
- 💯 **Free** - No registration required
- 🎤 **Voice Assistant** - Text and voice-powered AI assistant (Browser Native, Free)
- 🌙 **Dark Mode** - Beautiful dark theme support
- 📱 **Mobile First** - Responsive design for all devices
- ⚡ **PWA Ready** - Install as a Progressive Web App

## Tech Stack

- **Framework**: Astro 4 + React 18
- **Styling**: Tailwind CSS 3
- **Hosting**: Cloudflare Pages
- **TypeScript**: Full type safety
- **Voice**: Browser Native Speech Recognition & Synthesis (Web Speech API)

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or pnpm

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Project Structure

```
devtools-site/
├── public/          # Static assets
├── src/
│   ├── components/  # React & Astro components
│   ├── layouts/     # Page layouts
│   ├── lib/        # Utilities & data
```
